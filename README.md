# Student Management System Rest Service

This is a Student Management Service REST API, implemented CRUD operations.

### Tools Used :

- JDK 11 or higher
- Spring Boot 2.7.1
- Microsoft SQL Server
- IntelliJ IDEA for coding
- Azure Data Studio
